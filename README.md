# Мои конфиги

Репозиторий с моими конфигурационными файлами для различных сервисов и инструментов.

## Структура

Каждый сервис находится в отдельной директории и содержит собственный `README.md` с описанием, инструкцией по настройке и командами для использования.

## Сервисы

- [**HAProxy**](haproxy/README.md) — TLS-прокси для маршрутизации трафика по SNI с SSL-терминацией и автоматическим обновлением сертификатов.
- [**CrowdSec**](crowdsec/README.md) — настройка LAPI и нод CrowdSec в Docker Compose.
